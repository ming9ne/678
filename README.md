## 프로젝트소개
- 프로젝트 소개란 적기(이 문구는 나중에 지우기)
- **678 산책 메이트** 는 실시간 미세먼지 정보를 활용한 산책 메이트를 구하는 커뮤니티 입니다.
  사용자들에게 주변의 공원들을 추천하고, 해당 지역의 미세먼지 수준을 실시간으로 제공하여 건강하고 쾌적한 산책을 즐길 수 있도록 지원합니다. 함께 산책을 나가고 싶은 사람들을 매칭하여 새로운 친구를 만들고, 건강한 라이프스타일을 즐길 수 있는 서비스입니다.
